# 🛡️ AI Code Checker

An open-source, AI-powered code analysis platform built to identify bugs, stop secret leaks, and catch OWASP Top 10
vulnerabilities before they hit production. 

AI Code Checker bridges the gap between development and business teams by providing a **Risk Score Dashboard**
alongside an interactive **AI Chatbot Guide** capable of explaining vulnerabilities in both deep-dive technical terms and plain, jargon-free English.

### ✨ Key Features
* 📂 **Multi-Modal Ingestion:** Paste snippets, upload files, or scan entire GitHub repositories.
* 🛑 **Vulnerability & Leak Detection:** Catches OWASP Top 10 flaws and hardcoded API keys/secrets.
* 📊 **Risk Dashboard:** Color-coded metrics mapping your repository's overall health.
* 💬 **Dual-Audience Chatbot:** A context-aware assistant that guides users through secure code fixes.
